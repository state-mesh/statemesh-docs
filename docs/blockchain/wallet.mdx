title: "Wallet"

The StateMesh Wallet provides essential functionality for interacting with the StateMesh blockchain network. This official wallet offers both basic transaction capabilities and advanced network participation features.

## Access
Access the StateMesh Wallet through these official URLs:

The wallet is available online at:

- Testnet: [https://wallet-test.statemesh.org](https://wallet-test.statemesh.org)
- Mainnet: [https://wallet.statemesh.org](https://wallet.statemesh.org)


## Basic Functions

- View account balance and transaction history
- Send and receive funds, ERC-20 tokens, and other assets
- Manage multiple accounts

## Bridging Assets

- Bridge MESH and stablecoins to/from other blockchains
