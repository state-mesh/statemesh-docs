These brand resources serve as essential guidelines for anyone creating, supporting, or using StateMesh. They define our brand personality and help maintain consistent visual identity across all platforms and regions.

## Logo Usage Guidelines
When using StateMesh logos, please:

- Maintain the original proportions and shape
- Provide adequate padding around the logo
- Use only official color versions

Please avoid:

- Cropping or rotating the logo
- Using unauthorized color combinations
- Recreating the logo with different typefaces
- Adding shadows, transparency, or visual effects
- Altering the shape or proportions
